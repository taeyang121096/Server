리눅스 : 운영체제의 이름이다


배포판 : 리눅스에다 여러 소프트웨어를 섞어서 만든 운영체제를 리눅스 배포판이라고 한다.(커스터마이징) -> 데비안 계열, 레드햇 계열

대부분의 배포판은 gui와 cli를 모두 지원한다.
cli를 주로 사용한다. 모든 컴퓨터에서 사용가능하다.

컴퓨터의 소프트웨어와 하드웨어를 제어하는 것은 바로 os이다.

os는 컴퓨터의 하드웨어와 응용프로그램 상호작용,cpu,주기억 장치, 연산, 출력 데이터와 파일의 저장관리를 한다.

운영체제를 구성하는 2가지가 쉘과 커널이다.


커널 : 운영체제 내부에 있는 것으로 컴퓨터의 하드웨어를 제어한다.
하드웨어와 응용 프로그램 사이에서 인터페이스를 제공한다.
입출력을 관리하며 소프트웨어로부터 요청 system call을 하드웨어 system cal로 변환 해준다.


쉘 : 우리가 전달하는 스크립트를 커널이 이해하기 쉽게 해석해주는 역할을 한다.
실제로 명령어를 전달하고 결과를 전달받는 프로그램

터미널 : 인터페이스이다. 터미널은 gui 와 cli가 있다. 터미널은 주로 shell을 사용한다.
서버의 로컬 또는 원격으로 접속할 수 있는 콘솔을 구현한 소프트웨어

콘솔 :  터미널의 일종으로 시스템 제어를 위한 특수 목적 터미널이다.
서버의 로컬장치에서 직접 명령어를 작성할 수 있는 입출력 장치
